# Matheys.DEV

Portfólio pessoal — Matheus Silva (Next.js 15).

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deploy

Compatível com [Vercel](https://vercel.com) ou [Netlify](https://www.netlify.com): conecte este repositório e use o preset **Next.js**.

Base de layout inspirada no template [portfolio-2.0](https://github.com/Tajmirul/portfolio-2.0) (MIT).
